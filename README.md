## 🚀 Features

- 🎨 **Stunning UI** — Dark mode design with glassmorphism, glowing gradients & particle animations
- ⏳ **Live Countdown Timer** — Real-time countdown to the event date
- 📅 **Event Listings** — Browse all TechNova 2025 events and workshops
- 🧑‍🤝‍🧑 **Team Section** — Meet the organizers and crew
- 📍 **Location Section** — Venue details for the fest
- 📱 **Fully Responsive** — Works beautifully on all screen sizes
- ✨ **Smooth Animations** — Powered by Framer Motion

## 📦 Getting Started

### Prerequisites
- Node.js 18+
- npm / pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/Subhampops/technova-2025.git

# Navigate to the project directory
cd technova-2025

# Install dependencies
npm install
# or
pnpm install
```

### Run the development server

```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
